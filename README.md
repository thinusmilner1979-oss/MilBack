# MilBack

**MilBack** is a resilient system and network backup engine for Linux. It is designed for users who need a robust, "set-it-and-forget-it" solution that won't fail due to minor network blips or hardware timeouts.

### 🚀 Key Features
* **Resilient Engine:** Handles block-level read retries and stays alive during network share drops.
* **Multi-Profile Management:** Create and automate different backup sets for different needs.
* **Native Linux Integration:** Packaged as a `.deb` for a seamless experience on Mint, Ubuntu, and Debian.
* **Simple GUI:** A clean interface built with Python and PyQt6.

### 📦 Installation
1. Go to the [Releases](https://github.com/thinusmilner1979-oss/MilBack/releases) page.
2. Download the latest `milback_1.0.0_amd64.deb` file.
3. Double-click the downloaded file to install it using your system's package manager.
4. Launch **MilBack** from your application menu.

### 🛠 Tech Stack
* **Language:** Python 3
* **GUI Framework:** PyQt6
* **Platform:** Linux (Debian-based)
